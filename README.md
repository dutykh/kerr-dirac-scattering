# kerr-dirac-scattering
Companion code for "Two-channel Dollard scattering and partial-wave resonances for massive Dirac fields on Kerr" (Batić &amp; Dutykh): reproduces every table and figure — Jost-matching S-matrix benchmarks, machine-precision unitarity and conjugation certificates, and gate-certified quasi-bound resonances with a parameter-free Breit-Wigner fingerprint.
